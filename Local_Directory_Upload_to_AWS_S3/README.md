## ProjectTitle：Local_Directory_Upload_to_AWS_S3
ローカルディレクトリをAWS CLIコマンドを使用し、Amazon S3へ再帰的アップロードを行う

## Requirement
- Powershell

## Usage
1：Powershellプロンプトを起動する<br>
2：スクリプトを設置しているディレクトリまでcdする<br>
3：右記コマンドを実行する ⇒ *(cd先のカレントパス)/ Upload_To_S3.ps1<br>
